Извлеение сущностей и связей из текста на русском языке.
web-сайт , после ввода пользователем текста, выводит ему все сущности в тексте, а также схему того, как слова связаны между собой. 
Для этого используется библиотека Spacy и Natasha.
Также есть авторизация, которая позволяет просматривать историю запроов пользователя.
Возможность скачать файл с сущностями
